Protocolo HTTP

Se define como protocolo de transferencia de hipertexto

El funcionamiento se basa en solicitudes y respuestas

Cómo funciona el protocolo HTTP?

Primer paso

El cliente final establece conexión con un servidor e ingresa datos según el sitio web, a esto se le conoce como "request".

Segundo paso

Después el servidor web responde con la información que contenga, a esto se lo conoce como "response".
