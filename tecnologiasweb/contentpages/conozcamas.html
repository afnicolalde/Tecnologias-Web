Protocolo HTTP

Se define como protocolo de transferencia de hipertexto

El funcionamiento se basa en solicitudes y respuestas

Implementacion

El cliente final establece conexion ocn un servidor e ingresa datos segun el sitio web, a esto se le conoce como request

Despues el servidor web responde con la informacion que contenga, a esto se lo conoce como response
